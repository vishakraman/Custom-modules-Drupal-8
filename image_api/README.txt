

INTRODUCTION
------------
This module is only for the purpose to display https://pixabay.com/ website images on our site using owl carousel. The images are fetched from the API using this custom module. Place the block on any page, you get the carousel

Currenlty the carousel line is commented on the template file inside the module. 

Also there is another feature, please visit the url /pixa where you get a form.. place our module custom block in the /pixa url. so when the user selects any color that color image will be displayed.


REQUIREMENTS
------------
No requirements!
